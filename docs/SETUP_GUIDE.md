# セットアップ詳細手順 (Windows)

所要時間: 5分

## ステップ1. install.bat をダウンロード

このページの右上「Code」緑色ボタン → 「Download ZIP」 → 解凍 → `installer/install.bat` をダブルクリック

または、`installer/install.bat` を右クリック → 名前を付けて保存 で個別保存

## ステップ2. install.bat を実行

ダブルクリックすると黒いコマンドプロンプトが開いて以下が自動実行されます:

- ブラウザで「新規スプレッドシート」のタブが開く
- 別タブで「Apps Script のソースコード」が開く
- デスクトップに「タスク管理表.url」というショートカットが作成される

## ステップ3. スプレッドシートにコードを貼り付け

ブラウザで2つのタブが開いているはずです:

### 3-A. 「無題のスプレッドシート」タブ
1. 左上のスプレッドシート名をクリックして「タスク管理表」など好きな名前に変更
2. メニュー: `拡張機能` → `Apps Script` をクリック
3. 新しいタブで Apps Script エディタが開く
4. エディタ中央の `function myFunction() {}` などを **全選択して削除** (Ctrl+A → Delete)

### 3-B. 「Code.gs ソース」タブに切り替え
1. ブラウザで「raw.githubusercontent.com/.../Code.gs」と書かれたタブを開く
2. ページ全体を選択 (Ctrl+A) → コピー (Ctrl+C)

### 3-C. Apps Script エディタに戻る
1. さっき空にしたエディタに **貼り付け** (Ctrl+V)
2. 保存 (Ctrl+S) → スクリプト名を「タスク管理表」などに変更して OK

## ステップ4. 初期セットアップ実行

Apps Script エディタの上部:
1. 関数選択ドロップダウン (デフォルトは `onOpen`) を **`setupAll`** に変更
2. 隣の **「実行」(▶)** ボタンをクリック
3. 「権限の確認」ダイアログが出る → 「権限を確認」 → 自分のGoogleアカウントを選択
4. 「Google がこのアプリを審査していません」警告が出たら → 「詳細」 → 「無題のプロジェクト(安全ではないページ)に移動」 をクリック
   - これは自分で書いた(=自分でコピーした)スクリプトに対する警告です。安全
5. 「許可」をクリック
6. しばらく待つと「実行ログ」に「実行が完了しました」と出る
7. シートに6つのタブ(タスク一覧 / リマインドメールタスク / 企業リスト / 日報 / URL / 設定)が自動作成

## ステップ5. 動作確認

1. スプレッドシート画面に戻る (一度ページをリロードするとメニューが反映)
2. 上部メニューに新しく **「タスク管理」** が出ているはず
3. 「タスク管理」 → 「タスク追加」 をクリック → タスク一覧の2行目に新規行が追加される
4. 状況列のプルダウンで「進行中」を選ぶと黄色になることを確認
5. 期日列のセルをダブルクリックすると日付ピッカーが開くことを確認

## ステップ6. SMTG リマインドの設定 (任意)

1. **設定** シートを開く
2. カレンダーID は `primary` のままで OK (= 自分のメインGoogleカレンダー)
   - 別のカレンダーを使う場合は Google Calendar の左サイドバー → 該当カレンダー名にカーソルを当てて「3点メニュー」→「設定と共有」→ 下までスクロール →「カレンダーの統合」セクション →「カレンダーID」をコピー
3. SMTG キーワード も `SMTG` のままで OK (タイトルに含まれる予定が対象)
4. 「タスク管理」メニュー → 「SMTG リマインドを今すぐ作成」 でテスト実行
5. リマインドメールタスクシートに予定が反映されるか確認
6. `setupAll` 実行時に **毎朝6時の自動トリガー** が仕掛けられているので、以降は何もしなくても毎日6時にスキャンされます

## トラブルシューティング

### 「タスク管理」メニューが表示されない
→ ブラウザでスプレッドシートを一度リロード(F5キー)してください

### 「権限が拒否されました」エラー
→ Apps Script エディタで一度 `setupAll` を実行して権限付与してから再試行

### カレンダーが見つからない
→ 設定シートのカレンダーIDを確認 (`primary` か正しいカレンダーID)

### 行を追加してもプルダウンが効かない
→ メニューから一度「初期セットアップ」を再実行すると最大行までデータ検証が再適用されます

## 共有方法

完成したスプレッドシートは画面右上の **共有** ボタンから他の人に共有できます。

- メールアドレスを入力 → 「編集者」または「閲覧者」を選んで送信
- リンク取得 → 「リンクを知っている全員」 → 社内共有

共有相手も同じ Apps Script 機能(タスク追加・SMTG リマインドなど)を使えます。
