# task-board-sheets

Google スプレッドシート + Google Apps Script で動くタスク管理表。Google Cloud Console 一切不要。共有はスプレッドシートのリンク共有で完結。

## できること

- 5つのシートを1つのスプレッドシートに集約
  1. **タスク一覧** タスク追加日 / 企業名 / タスク内容 / 期日 / 優先度 / 詳細 / 状況
  2. **リマインドメールタスク** カレンダーの SMTG 予定の前営業日に自動でリマインドが入る
  3. **企業リスト** 企業名 / ユニット / アサイン日 / 支援期間 / 議事録URL / 支援責任者 / 支援担当者
  4. **日報** 日報フォーマットコピー + 過去日報の蓄積
  5. **URL** よく使うURLを集めるシート
- 「タスク追加」で先頭行に新規行を挿入(上に追加)
- 状況プルダウン (未着手 / 進行中 / 依頼中 / 完了) で自動色分け
- 期日列は日付ピッカー (ダブルクリックでカレンダーが開く)
- SMTG リマインドは **毎朝6時に自動生成**(土日・日本祝日スキップで前営業日を期日に設定)
- 設定シートでカレンダーID・キーワード・先読み日数・日報テンプレートを変更可能
- 改善アイデアの投稿フォームを内蔵

## セットアップ

### 方法A: ブラウザだけで完結 (推奨、Windows のセキュリティ警告ナシ)

GitHub Pages にホストしたインストーラーページを開きます:

**https://yuuto22009911-hash.github.io/task-board-sheets/**

(GitHub Pages 有効化後にアクセス可能)

ページに表示される手順に従って、ボタンをクリックするだけでセットアップできます。
Apps Script のコードもページ内の「コードをコピー」ボタンでクリップボードに入ります。

### 方法B: HTML ファイルをダウンロードして開く (オフライン)

1. [installer/install.html](installer/install.html) を右クリック → 「リンク先を別名で保存」 → 任意の場所に保存
2. 保存した `install.html` をダブルクリック → 既定ブラウザで開く
3. ページ内の手順通りに進める

HTML ファイルは Windows のセキュリティでブロックされません。

### 方法C: バッチファイル (Windows、セキュリティ警告が出る場合あり)

1. [installer/install.bat](installer/install.bat) をダウンロード → ダブルクリック
2. 「Windows によって PC が保護されました」が出たら → 「詳細情報」 → 「実行」

### 方法D: 完全手動 (どの OS でも共通)

1. ブラウザで `https://sheets.new` を開いて新規スプレッドシート作成
2. メニュー `拡張機能 → Apps Script` をクリック
3. 開いたエディタのコードを全削除
4. このリポジトリの [apps-script/Code.gs](apps-script/Code.gs) の中身を全部コピーして貼り付け
5. 保存 (Ctrl+S または Cmd+S)
6. 関数選択ドロップダウンから `setupAll` を選んで「実行」
7. 初回のみ権限確認ダイアログ → 「許可」をクリック
8. 完了したらスプレッドシートに戻ると上部メニューに「タスク管理」が表示されます

## 共有方法

スプレッドシートの右上 **共有** ボタンから:
- メールアドレスを指定して共有(編集者/閲覧者の権限選択可)
- リンクを知っている人全員に共有(社内共有向け)

スプレッドシートを共有すれば、共有相手も同じ Apps Script 機能を使えます。

## カレンダー設定

`設定` シートで:
- **カレンダーID**: `primary`(自分のメインカレンダー)を既定値に。別カレンダーを使う場合は Google Calendar の「設定と共有 → カレンダーの統合 → カレンダーID」から取得して入力
- **SMTG キーワード**: 既定 `SMTG`(部分一致・大文字小文字区別なし)
- **先読み日数**: 何日先まで予定をスキャンするか
- **日報テンプレート**: 日報の雛形(`{DATE}` が今日の日付に置き換わる)

## 自動実行トリガー

`setupAll` を実行すると自動で **毎朝6時にリマインド作成** のトリガーが仕掛けられます。
手動で再設定したい場合は Apps Script エディタのトリガーアイコン(時計マーク)から確認できます。

## ライセンス / 開発フロー

このリポジトリは GitHub-first 運用です:
- 機能追加は `feat/xxx` ブランチで PR を作成
- CI なし(Apps Script は GAS 上で実行されるためローカルテスト対象外)
